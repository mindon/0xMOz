① 寫 index.html — Present 入口頁

展示頁是整個 Present 的入口，定義 slide 列表、加載順序與導航結構。 每一個 <slide> 都是滾動快照中的一頁。

➊ 最小骨架

<!DOCTYPE html>
<html lang="zh_CN">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>我的演示</title>
    <link href="./assets/present.css" rel="stylesheet">
    <script type="module" src="./present.stjvtw1s.js"></script>
  </head>
  <body>
    <div class="scroll-snap">
      <slide class="slide-center" topic="封面" data-src="cover.html" instantly></slide>
      <slide class="slide-center" topic="第二頁" data-src="page2.html"></slide>
      <slide class="slide-center pop" topic="連接" data-src="more.html"></slide>
    </div>
    <div id="snap-points"></div>
  </body>
</html>

❝ .scroll-snap 容器啟用滾動快照； #snap-points 是右側自動生成的目錄錨點容器。

➋ <slide> 屬性速查

屬性 說明（*為內容頁面使用） 示例
topic 當前頁標題，自動生成右側導航與 URL hash topic="封面"
data-src 該 slide 的內容頁路徑（相對於 slides/） data-src="cover.html"
data-lazy * 對應 lazies.slide(key, fn) 註冊的初始化函數鍵 data-lazy="my-init"
instantly 可选布爾屬性。設置後立即加載，不等待滾動到該頁 instantly
class="slide-center" 內容垂直水平居中 class="slide-center"
class="pop" * 啟用點擊彈框（需配合 data-pop） class="slide-center pop"
data-pop * 彈框元素選擇器（一般為 #xxx） data-pop="#info"
data-spot * 固定彈框錨定位置（可選） x,y，支持百分比/負值 data-spot="0.5,0.3"
web-or-not * 啟用網絡探測，配合 .web-yes / .web-no 分支顯示 web-or-not

➌ 路徑約定

data-src="foo.html" 會被 present.stjvtw1s.js 解析為 fetch("slides/foo.html")；
因此 index.html 必須放在 slides/ 的父目錄，而所有 slide 內容頁放在 slides/ 之下。 

your-project/
├─ index.html        ← 展示頁入口
├─ assets/present.css
├─ present.stjvtw1s.js
└─ slides/
   ├─ cover.html     ← <slide data-src="cover.html">
   ├─ page2.html
   └─ more.html

➍ 直接打開單頁

所有 slide 內容頁都是完整可獨立打開的 HTML，比如： /slides/cover.html 可直接訪問， present.stjvtw1s.js 會自動切換到「單頁模式」並添加 ❝ HOME ❞ 跳回按鈕。
