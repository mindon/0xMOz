什麼是 ∅M〇ᶻ Present？

一套基於原生Web標準（scroll-snap + popover + importmap）的幻燈片框架。
每個 slide 都是獨立 HTML，可單獨打開、分享，亦可串聯成完整 Present。

🧩 Slide 獨立展示
每頁是完整 HTML，可直接打開 /slides/xxx.html
💬 信息彈框（Popover）
原生 popover + 自動定位箭頭
🪄 懶加載 lazies.slide
滾動到當前頁才注入腳本/樣式
📦 Import Map 合併
各 slide 自帶 importmap，自動合入主文檔
🧭 Hash 導航
支持瀏覽器前進/後退、深鏈接
📜 Scroll-Snap 切換
順滑、原生、零依賴
點擊本頁任意處 → 查看更多介紹（彈框示例）

∅M〇ᶻ Present 的設計哲學

  • ✦ 原生優先：直接使用瀏覽器標準 API（Popover / Scroll-Snap / IntersectionObserver / Importmap），不依賴任何框架。
  • ✦ 漸進式呈現：頁面按需加載，未滾動到的 slide 不會觸發網絡請求。
  • ✦ 可獨立分享：任意 slide 對應一個 HTML 文件，URL 可單獨打開、嵌入或截圖。
  • ✦ 少即是多：核心邏輯集中在 present.n8e91kk6.js，無需學習額外 DSL。
  • ✦ 對 AI 友好：純 HTML/JSON 描述結構，便於 AI Agent 自動生成或編輯。
